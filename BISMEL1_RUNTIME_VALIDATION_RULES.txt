- position manager acts only on reconciled broker and local state, then writes safe management summaries

Duplicate protection
- scheduler skips duplicate bar-close windows
- execution skips duplicate recent broker actions and closes stale running execution rows before retry

Blocked state wording
- blocked states stay summary-first: broker not ready, strategy not mapped, execution requires review, or position management requires review

Safe customer and admin visibility

Hidden strategy handling
